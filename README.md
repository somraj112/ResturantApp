
# 🍽️ Gericht Restaurant Website

A modern and responsive restaurant website built using React.  
This project showcases a stylish UI for a fine dining experience, including sections like About, Menu, Chef's Word, Awards, and more.

---

## 🚀 Features

- 🍴 Elegant restaurant landing page  
- 📖 About & History sections  
- 👨‍🍳 Chef's word and specialties  
- 🏆 Awards & recognition section  
- 📱 Fully responsive design  
- ⚡ Built with React for fast performance  

---

## 🛠️ Tech Stack

- React.js  
- CSS / SCSS  
- JavaScript (ES6+)  

---

## 📦 Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/somraj112/ResturantApp.git
cd ResturantApp
npm install
npm start
````
